二値分類のサンプルその1
=======================

パーセプトロンといえばXOR、ということで、XORを学習。

データファイルはSVMlight形式。

	0 1:0 2:0
	1 1:1 2:0
	1 1:0 2:1
	0 1:1 2:1

ラベルは0または1に。

訓練データは数が少なくても大丈夫。(整数倍して繰り返し使う)
